Vacation Planner App

Welcome to the official deployment page for the Vacation Planner mobile application (D424 Capstone Project).

Project Overview

This Android application allows users to manage their vacations and excursions with the following features:

  • Secure Login: Industry appropriate security features with authenticated access.
  • Vacation Management: Create, edit, and track your upcoming trips.
  • Excursion Tracking: Manage activities within your vacations with database-backed category selection.
  • Advanced Reporting: Generate parameterized reports filtered by specific date ranges.
  • Alerts: Set notifications for important vacation and excursion dates.
